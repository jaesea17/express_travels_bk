## Description
The backend to the Express travels app. An app the helps clients book bus tickets to different parts of the country

## Installation

```bash
$ npm install
```

## Running the app
Make sure you have docker installed and running.

```bash
# development
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
